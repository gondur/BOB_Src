Daily Intelligence Summary

This is the Intelligence Summary for the day. The list of Intelligence Topics available to you can be filtered using the radio buttons at the top of the Dialogue:

Target/General

If Target is ticked then only those intelligence topics associated with a particular location or target are listed.

If General is ticked then all intelligence topics are listed.

Latest/Priority

If Latest is ticked then only those intelligence topics generated in the last session are listed.

If Priority is ticked then all priority intelligence topics are listed. The system is only capable of recording a finite set of topics and so old topics are automatically removed.

You can use the three buttons at the bottom of the Dialogue to get further information about the highlighted Intelligence Topic:

Notes

Click on this button to get notes on the highlighted topic

Footage

Click on this button to get film footage on the highlighted topic. This is only available for key events during the campaign.

Intelligence

Click on this button to open up the Dossier for the Location/Target referred to in the highlighted topic.

If a button is greyed-out then that option is not available for the highlighted Intelligence Topic.
