Tasks

The Task Dialogue contains detailed information about the aircraft of the current Wave. The details for one Group of the Wave are displayed. Use the Tabs at the top of the Dialogue to view the details of the other Groups.

The first line shows the Squadron Information, for instance B26(1/2). This indicates that the current Group consists of one flight of B26 aircraft. The 2 means that up to two flights of aircraft could be allocated to this group. Click on the box to see what other alternative aircraft could be used for this Current Group.

The next section of the Dialogue is concerned with the detailed orders that can be issued to the Groups. For the main duty it is possible to specify:
Attack Method

Attack Pattern

Group Formation

The Escort Position combo is not relevant.


For the Escort Groups, it is possible to specify:
Radio Silence

Group Formation

Escort Position

The last section of the Dialogue is concerned with giving orders to the individual flights of the Group.

The box next to the Flight designation shows the payload required for the flight. Click on the box to see what other payloads could be defined for this Current Flight.

Extra Targets

The two boxes next to the Target designation show the flight’s targets. It is possible to define a main and a secondary target for each flight. When a flight is engaged in a patrol duty then these two targets define the extent of the patrol area. Click on a target box to change the allocated target. You will be presented with a Dialogue requesting you to select a target. Click on an item icon on the map to define the target. This Dialogue has two buttons:

Reset

Reset the target to the Mission’s Main Target.

Cancel

Cancel the request to change the target.

After selecting extra targets, the new waypoints will be detailed in the Routes directory.
