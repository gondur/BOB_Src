Filters Toolbar

The Filters Toolbar is used to control the icons and lines that are drawn on the map. There are three sets:

Blue Set

This set of icons controls the display of the UN Forces and assets. The silver-backed icon on the left can be used to toggle on and off all UN Forces and Assets.

Red Set

This set of icons controls the display of the Communist Forces and assets. The red set of icons works in the same way as the blue set.

Miscellaneous Set

The top left icon toggles the main waypoints.

The bottom left icon toggles the waypoints in the target zone.

The top right icon toggles between displaying all the route lines and just the highlighted route.

The bottom right icon toggles the Front Line.
