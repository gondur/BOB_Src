H Directory: All header files and generated header files
=========== 

This directory contains most of the header files in the project.
It also contains generated files (.g extension)

Concatinated source file headers
--------------------------------
We have found that the program compiles significantly faster if we concatinate
a number of source files together rather than compiling each file seperately.
Unfortunately, some of the interdependencies in the header files break down, so
we have these corrective header files available:

_3D.H		Force correct loading of header files in _3D.CPP	
_AFX.H		Force correct loading of header files in _AFX.CPP
